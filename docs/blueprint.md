# **App Name**: Restake Oasis

## Core Features:

- Data Dashboard: Display key metrics like TVL, total restakers, and APR using clear charts and counters.
- Partner Showcase: Visually showcase the ecosystem partners with an animated ticker to increase brand awareness and perceived value.
- One-Click Restaking Simulation: Provide a simulation where users can visualize restaking rewards, incorporating estimated APR and restaking period length. The LLM tool will assist in providing more contextual parameters for the simulation to run more effectively. The goal of the simulator is to provide the user with additional knowledge to allow them to determine if they are comfortable performing a one-click restake action.
- Audited Security Visuals: Visually represent the security measures, displaying the audit reports, and providing assurance to users.

## Style Guidelines:

- Primary color: Teal (#2DD4BF) for trustworthiness, and growth. Avoid using excessive Teal to improve accessibility and aesthetics.
- Background color: Light gray (#F9FAFB) to provide a clean and modern backdrop.
- Accent color: Soft Blue (#94A3B8) as a secondary color, to ensure consistent visibility for supplementary text and interactive elements.
- Body font: 'Inter', a sans-serif font for clear, modern text.
- Headline font: 'Clash Display', a sans-serif font used for headlines.
- Use icons from Lucide-React for a consistent and modern look.
- Incorporate subtle animations, such as the scanner, utility ring rotation, and parallax tunnel, to improve engagement without being distracting.
